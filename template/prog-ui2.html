Connaissances

Théoriques (quelles sont les notions que vous avez retenues) ?

Nous avons appris la programmation orientée objet (POO) pour les interfaces. Nous avons utilisé des requêtes asynchrones pour mettre à jour ces interfaces. Des concepts avancés en programmation côté client, tels que le routage et les gestionnaires d'état, ont également été abordés.

Pratiques (quels sont les logiciels que vous avez appris) ?

Nous avons appris à utiliser le langage JavaScript.

Conditionnelles (quels sont les projets que vous avez réalisés) ?

  • Nous avons réalisé quelques exercices (défis) pour comprendre les nouveaux concepts.
  • Nous avons créé un compteur dont la valeur augmente en appuyant sur un bouton.
  • Nous avons réalisé un exercice de "drag and drop" qui consistait à prendre un carré, cliquer dessus et le déplacer.
  • Nous avons réalisé un TP1 qui consistait à répondre à 7 questions en utilisant les concepts nouvellement appris.
  • Nous avons réalisé un exercice qui consistait à créer un bouton de type "toggle".
  • Nous avons réalisé un exercice de "Tic Tac Toe".
  • Nous avons réalisé un exercice de "Jeux de société" qui consistait à cliquer sur une image de jeu pour obtenir des informations sans recharger la page.
  • Enfin, pour conclure le cours, nous avons réalisé un projet de synthèse qui consistait à reproduire un lecteur de musique similaire à Spotify.
