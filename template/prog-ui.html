Connaissances

Théoriques (quelles sont les notions que vous avez retenues) ?

Dans ce cours d'introduction à la programmation côté client, vous apprendrez à manipuler les composantes d'une page Web, à y ajouter de l'interactivité et à traiter des données dynamiques. De plus, vous serez initié aux principes de la programmation orientée objet (POO).

Pratiques (quels sont les logiciels que vous avez appris) ?

  • Visual Studio Code (avec Vue.js)

Conditionnelles (quels sont les projets que vous avez réalisés) ?

  • Exercice 1 - Création d'une page qui ajoute une nouvelle image lors d'un clic sur un lien
  • TP1 - Répondre aux 8 questions qui sont sous forme de petits exercices
  • TP2 - Créer un formulaire qui permet de répéter une phrase un certain nombre de fois avec un choix de couleur pour le texte et la couleur de fond sans recharger la page
  • Examen de mi-session - Création d'un carré par JS qui bouge de façon aléatoire lorsqu'on clique dessus, ainsi que des boutons pour changer sa couleur
  • Examen synthèse - Créer une page Web qui permet de personnaliser une émoticône en temps réel sans devoir actualiser la page
