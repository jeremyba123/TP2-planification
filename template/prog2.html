Connaissances

Théoriques (quels sont les notions que vous avez retenues)?

Ce deuxième cours de programmation côté serveur a pour objectif d’apprendre à utiliser une base de données. Plus précisément, vous apprendrez à vous connecter à une base de données par programmation, à créer des structures de données, à effectuer des opérations et à traiter les données afin de produire une réponse à une requête HTTP

Pratiques (quels sont les logiciels que vous avez appris)?

  • Visual Studio Code
  • PHPMyAdmin
  • XAMPP

Conditionnelles (quels sont les projets que vous avez réalisés)?

  • TP1 - Création d'une base de données respectant le devis fourni
  • TP2 - Création d'une base de données ayant pour thème l'actualité, des pages pour ajouter des données ainsi qu'une page pour voir les actualités sélectionnées
  • Examen de mi-session pratique - Création d'une base de données ayant pour thème une compétition sportive
  • Examen synthèse pratique - Création d'une base de données et d'une page web ayant pour thème une to-do list
